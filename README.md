# android-without-libraries

Ejemplo de una app que consume el [API de Meetup](https://www.meetup.com/es-ES/meetup_api/) usando mvp & clean architecture 

Dependencias

- Android Support, Android Design

- Retrofit 2

- GSon

- Dagger 2

- Picasso, Glide

- Renderers
